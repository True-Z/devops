## 创建并配置 ssh key

### 1. 检查本机是否存在 ssh key

```sh
cd ~/.ssh
ls
```

### 2. 生成 ssh key（已存在直接第3步）

```sh
ssh-keygen -t rsa -C "643345861@qq.com"

# 后续回车即可
```

### 3. 获取 ssh key 内容

```sh
cd ~/.ssh
cat id_rsa.pub
```

### 4. 在 git 托管网站上添加公钥

按不同网站规定添加即可。

### 5. 验证是否成功

```sh
ssh -T git@gitXXX.com

# 成功会有相应的提醒
```
